TrueRandom
==========

TrueRandom is a plugin for Simba which allows developers to generate truly random numbers queried from Random.org.

TrueRandom is built ontop of an API made by Doug Hague, released under the GNU license.

Dependent Libraries:
====================

TrueRandom's API requires cURL, it is therefore distributed with this plugin.
